#epcrd

`epcrd` is the planned software companion to the Enhanced PCRD (Politician-Characteristic Regression Discontinuity) framework.

The package is in development and will provide tools for:
- constructing predicted-close subsamples,
- implementing vote-margin neutrality diagnostics,
- sensitivity analysis for practical neutrality,
- workflow support for disciplined focal-trait attribution.

Initial R package release is forthcoming. A Stata implementation may follow.

## Status
Working paper in circulation; software not yet released.
